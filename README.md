# K-Soccer - Football Object Detection Project

The goal of this project is to analyze soccer matches using YOLO and image proccessing with OpenCV. 
You can check the project walkthrough in [explanation.ipynb](https://github.com/davidhoruna/K-Soccer/blob/main/explanation.ipynb)  ðŸš€

## Features

- **Player Detection**: Detects players on the field using YOLO.
- **Team Assignment**: Assigns players to teams based on jersey colors.
- **Ball Possession Tracking**: Tracks which team has possession of the ball.
- **Frame Processing**: Processes video frames to extract relevant information.
- **Debugging**: Provides visual debugging information on the processed frames.
